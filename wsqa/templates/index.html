Water Surface Quality API

This Application Programming Interface (API) gives software developers access to the Water Surface Quality measurments done in all the Kosovo rivers. This data is compiled by Kosovo Agency for Environment Protection (KEPA).

We care about sharing and improvements, so the software behind this API is free and open under the GNU GPL v2.0 license. Anyone who distributes the code or a derivative work is required to make the source available under the same terms.


Disclaimer

This API is an alpha version. This means that it is still under development and may be subject to bugs. You can help by contacting us to report issues or contribute fixes and improvements to the code base.


Getting Data

Retrieving data from the API is done through RESTful HTTP GET Requests.


All the measuremnts done in the Kosovo rivers

Get all the measurments done throughout the regions rivers:
/measurements


All the averages

Get all the average measurements of all the stations on a given river:
/aggregate/average/river/<string:river_slug>

Get all the average measurements of all the stations on a given river based on a given year and month:
/aggregate/average/river/<string:river_slug>&year=int?month=int

Get all the average measurements of each station on a given river:
/aggregate/average/river/<string:river_slug>/stations

Get all the average measurements of every station:
/aggregate/average/stations


Sample Requests

Get all the measurements done in every river:
{{ config.BASE_PATH }}/measurements


Get all the measurements done in every river on a given year and month:
{{ config.BASE_PATH }}/measurements&year=2013?month=06


Get all the average measurements of all the stations in Lumbardhi i Pejes:
{{ config.BASE_PATH }}/aggregate/average/river/lumbardhi-i-pejes


Get all the average measurements of each station in Lumi i Erenikut:
{{ config.BASE_PATH }}/aggregate/average/river/lumi-erenik/stations


Get all the average measurements of every station in Kosovo:
{{ config.BASE_PATH }}/aggregate/average/stations
